# Intégration Sellsy-Airtable

Ce projet permet de synchroniser les emails envoyés depuis Sellsy vers une base de données Airtable.

## Prérequis

- Python 3.10 ou supérieur (via Anaconda)
- Compte Sellsy avec accès API
- Compte Airtable
